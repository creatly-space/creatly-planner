# Creatly Planner

Project planner with board, timeline, calendar, and list views. Built with React + Supabase for real-time sync across devices.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the `creatly-planner` repo
4. Click Deploy (no config needed, Vercel auto-detects Vite)
5. Your app will be live at `your-project.vercel.app`

## iPad Setup

1. Open the Vercel URL in Safari on iPad
2. Tap the Share button → "Add to Home Screen"
3. It works like a native app with full-screen mode

## Local Development

```bash
npm install
npm run dev
```
